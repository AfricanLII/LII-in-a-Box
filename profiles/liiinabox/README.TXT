LII-in-a-Box
============

The LII Drupal site platform.

Build using Drush:
	$ git clone git@bitbucket.org:spinningyourweb/liiinabox.git
	$ cd liiinabox
	$ drush make build-liiinabox.make /tmp/test_lii_box_site

If you look at the new Drupal install (/tmp/test_lii_box_site) you will see the liiinabox profile in /profiles which contains all the modules etc.
