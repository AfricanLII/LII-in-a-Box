LII-in-a-Box
============

The LII Drupal site platform.

Build using Drush:
        $ Go to the document root directory
	$ git clone yourusername@https://github.com/africanlii/LII-in-a-Box
	$ cd LII-in-a-Box or rename to folder to prefered name
	$ drush make build-liiinabox.make . or to any directory eg /tmp/test_lii_box_site

If you look at the new Drupal install (LII-in-a-Box) you will see the liiinabox profile in /profiles which contains all the modules etc.
